# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of this addon.

## Reporting a Vulnerability

Please do not report security issues in public GitHub issues.

Use one of these channels instead:

1. GitHub Security Advisory (preferred):
   https://github.com/FriendsOfREDAXO/tinymce/security/advisories/new
2. If advisory creation is not possible, open a private contact via maintainers:
   https://github.com/FriendsOfREDAXO/tinymce/issues

Please include:

- A clear description of the vulnerability
- Reproduction steps or proof of concept
- Affected versions
- Potential impact

We will acknowledge valid reports as quickly as possible and coordinate a fix and release.
